Thiết Kế Web

Học cách xây dựng website từ con số 0 với HTML, CSS và JavaScript - ba công nghệ nền tảng của web.

Bộ ba công nghệ web

  • HTML - Cấu trúc nội dung
  • CSS - Định dạng giao diện
  • JavaScript - Xử lý tương tác

HTML - Hypertext Markup Language

HTML là ngôn ngữ đánh dấu để tạo cấu trúc cho trang web.

Cấu trúc cơ bản

<!DOCTYPE html> <html lang="vi"> <head> <meta charset="UTF-8"> <title>Trang web đầu tiên</title> </head> <body> <h1>Xin chào!</h1> <p>Đây là trang web đầu tiên.</p> </body> </html>

CSS - Cascading Style Sheets

CSS giúp trang web trở nên đẹp mắt với màu sắc, font chữ, layout...

/* Định dạng cơ bản */ body { font-family: 'Arial', sans-serif; background-color: #f0f0f0; margin: 0; padding: 20px; } h1 { color: #333; text-align: center; } .button { background: #007bff; color: white; padding: 10px 20px; border-radius: 5px; }

JavaScript

JavaScript giúp trang web tương tác với người dùng.

// Hiển thị thông báo alert("Chào mừng đến Tin Học Đường!"); // Xử lý sự kiện click document.getElementById("myBtn").addEventListener("click", function() { alert("Bạn đã click!"); }); // Thay đổi nội dung document.getElementById("demo").innerHTML = "Nội dung mới";
