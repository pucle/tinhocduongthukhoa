Thuật Toán & Cấu Trúc Dữ Liệu

Thuật toán và Cấu trúc dữ liệu là nền tảng của khoa học máy tính. Đây là kiến thức cốt lõi cho mọi lập trình viên và học sinh thi HSG Tin học.

Tầm quan trọng

Giải quyết bài toán hiệu quả không chỉ là viết code chạy đúng, mà còn phải chạy nhanh và tiết kiệm bộ nhớ.

Stack (Ngăn Xếp)

Stack là cấu trúc dữ liệu theo nguyên tắc LIFO (Last In First Out) - phần tử vào sau ra trước.

Ứng dụng:

  • Quản lý lời gọi hàm (call stack)
  • Kiểm tra ngoặc hợp lệ
  • Tính toán biểu thức hậu tố
  • Duyệt đồ thị (DFS)
// Cài đặt Stack bằng mảng int stack[1000]; int top = -1; void push(int x) { stack[++top] = x; } int pop() { return stack[top--]; } bool isEmpty() { return top == -1; }

BFS - Tìm Kiếm Theo Chiều Rộng

BFS duyệt đồ thị theo từng tầng, từ đỉnh xuất phát ra các đỉnh lân cận.

Ứng dụng:

  • Tìm đường đi ngắn nhất (đồ thị không trọng số)
  • Kiểm tra tính liên thông
  • Tìm các thành phần liên thông
void BFS(int start) { queue<int> q; q.push(start); visited[start] = true; while (!q.empty()) { int u = q.front(); q.pop(); for (int v : adj[u]) { if (!visited[v]) { visited[v] = true; q.push(v); } } } }

Quy Hoạch Động - Fibonacci

Quy hoạch động giải quyết bài toán bằng cách chia thành các bài toán con và lưu trữ kết quả.

// Cách 1: Top-down với Memoization int memo[1000]; int fib(int n) { if (n <= 1) return n; if (memo[n] != -1) return memo[n]; return memo[n] = fib(n-1) + fib(n-2); } // Cách 2: Bottom-up int fib(int n) { int dp[n+1]; dp[0] = 0; dp[1] = 1; for (int i = 2; i <= n; i++) dp[i] = dp[i-1] + dp[i-2]; return dp[n]; }
