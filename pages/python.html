Học Python

Python là ngôn ngữ lập trình phổ biến nhất thế giới, dễ học và có ứng dụng rộng rãi từ web, data science đến AI.

Tại sao nên học Python?

  • Cú pháp đơn giản, dễ đọc
  • Thư viện phong phú
  • Cộng đồng lớn, nhiều tài liệu
  • Ứng dụng đa dạng: Web, AI, Data Science, Automation

Bắt Đầu Với Python

Hello World

# Chương trình Python đầu tiên print("Hello, World!") print("Xin chào Tin Học Đường!")

Biến và Kiểu dữ liệu

# Các kiểu dữ liệu cơ bản name = "Nguyễn Văn A" # String age = 16 # Integer gpa = 8.5 # Float is_student = True # Boolean # In thông tin print(f"Họ tên: {name}") print(f"Tuổi: {age}") print(f"Điểm TB: {gpa}")

Cấu Trúc Điều Khiển

If-Else

diem = 8.5 if diem >= 8.5: print("Giỏi") elif diem >= 6.5: print("Khá") elif diem >= 5.0: print("Trung bình") else: print("Yếu")

Vòng lặp For

# In các số từ 1 đến 10 for i in range(1, 11): print(i) # Duyệt danh sách subjects = ["Toán", "Lý", "Hóa", "Tin"] for subject in subjects: print(f"Môn: {subject}")

Hàm (Function)

# Định nghĩa hàm def tinh_tong(a, b): """Tính tổng 2 số""" return a + b def xep_loai(diem): """Xếp loại học sinh""" if diem >= 8.5: return "Giỏi" if diem >= 6.5: return "Khá" if diem >= 5.0: return "TB" return "Yếu" # Sử dụng hàm print(tinh_tong(5, 3)) # Output: 8 print(xep_loai(8.7)) # Output: Giỏi

List và Dictionary

# List - Danh sách scores = [8.5, 9.0, 7.5, 8.0] print(f"Điểm TB: {sum(scores)/len(scores)}") # Dictionary - Từ điển student = { "name": "Nguyễn Văn A", "class": "11A1", "gpa": 8.5 } print(f"Học sinh: {student['name']}")

Bài Tập Thực Hành

Dễ

  • Tính tổng các số từ 1-N
  • Kiểm tra số chẵn/lẻ
  • In bảng cửu chương

Trung bình

  • Kiểm tra số nguyên tố
  • Tính giai thừa
  • Đảo ngược chuỗi

Khó

  • Giải phương trình bậc 2
  • Sắp xếp danh sách
  • Đọc/ghi file
