Thuật Toán Sắp Xếp

Ứng dụng về sắp xếp có ở khắp mọi nơi:

  • Một danh sách lớp với các học sinh được sắp xếp theo thứ tự bảng chữ cái.
  • Một danh bạ điện thoại.
  • Danh sách các truy vấn được tìm kiếm nhiều nhất trên Google.

Thuật toán sắp xếp cũng được dùng kết hợp với những thuật toán khác, như tìm kiếm nhị phân, thuật toán Kruskal để tìm cây khung nhỏ nhất của đồ thị.

Vì sao phải học nhiều thuật toán sắp xếp?

Khi code, bạn chỉ cần biết cài một thuật toán sắp xếp là đủ. Hoặc nếu bạn code C++ hay Java, bạn chỉ cần biết cách gọi thư viện. Tuy nhiên, các thuật toán sắp xếp khác nhau cho ta nhiều ý tưởng hay và độc đáo - điều này vô cùng hữu ích khi các bạn học các thuật toán khác.

Những điểm cần chú ý

Khi so sánh giữa các thuật toán này với nhau, có nhiều vấn đề phải quan tâm:

  1. Thời gian chạy: Đối với các dữ liệu rất lớn, các thuật toán không hiệu quả sẽ chạy rất chậm.
  2. Bộ nhớ: Các thuật toán nhanh đòi hỏi đệ quy sẽ có thể phải tạo ra các bản copy của dữ liệu.
  3. Độ ổn định (stability): Các phần tử có giá trị bằng nhau có giữ nguyên thứ tự không?

1. Sắp xếp nổi bọt (Bubble Sort)

Đây là thuật toán cơ bản nhất cho việc sắp xếp.

Ý tưởng:

  • Xét lần lượt các cặp 2 phần tử liên tiếp. Nếu phần tử đứng sau nhỏ hơn phần tử đứng trước, ta đổi chỗ 2 phần tử.
  • Lặp lại đến khi không còn 2 phần tử nào thỏa mãn.

Ưu điểm: Code đơn giản, dễ hiểu. Không tốn thêm bộ nhớ.

Nhược điểm: Độ phức tạp O(N²), không đủ nhanh với dữ liệu lớn.

for (int i = 0; i < n; i++) { for (int j = 0; j < n - 1; j++) { if (a[j] > a[j+1]) { swap(a[j], a[j+1]); } } }

2. Sắp xếp nhanh (Quick Sort)

Một trong những thuật toán sắp xếp hiệu quả nhất trong thực tế.

Ý tưởng:

  • Chọn một phần tử làm chốt (pivot).
  • Phân hoạch mảng thành 2 phần: nhỏ hơn pivot và lớn hơn pivot.
  • Gọi đệ quy cho 2 phần mảng.

Độ phức tạp: Trung bình O(N log N), xấu nhất O(N²).

void quickSort(int a[], int left, int right) { if (left >= right) return; int pivot = a[(left + right) / 2]; int i = left, j = right; while (i <= j) { while (a[i] < pivot) i++; while (a[j] > pivot) j--; if (i <= j) { swap(a[i], a[j]); i++; j--; } } quickSort(a, left, j); quickSort(a, i, right); }

Tìm Kiếm Nhị Phân

Tìm kiếm nhị phân là thuật toán tìm kiếm hiệu quả trên mảng đã được sắp xếp.

Ý tưởng:

  • So sánh phần tử cần tìm với phần tử giữa mảng.
  • Nếu bằng → tìm thấy.
  • Nếu nhỏ hơn → tìm ở nửa trái.
  • Nếu lớn hơn → tìm ở nửa phải.

Độ phức tạp: O(log N)

int binarySearch(int a[], int n, int x) { int left = 0, right = n - 1; while (left <= right) { int mid = (left + right) / 2; if (a[mid] == x) return mid; if (a[mid] < x) left = mid + 1; else right = mid - 1; } return -1; // Không tìm thấy }

Bài Tập Thực Hành

Dễ

  • Sắp xếp mảng tăng dần
  • Tìm phần tử lớn nhất
  • Đếm số lần xuất hiện

Trung bình

  • Tìm cặp số có tổng = K
  • Merge 2 mảng đã sắp xếp
  • Tìm phần tử xuất hiện nhiều nhất

Khó

  • Quick Sort không đệ quy
  • Counting Sort
  • Radix Sort
